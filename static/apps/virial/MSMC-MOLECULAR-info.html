MSMC for moleclues

The MSMC for moleclues page runs a Mayer-sampling Mnte Carlo simulation for a rigid molecule. Monte Carlo moves are used to translate and rotate molecles without altering the bond lengths and angles. The nature of the simulation is very similar to the MSMC simulation used for single-atom MSMC and the MSMC information is also applicable to the molecular approach.

Molecular model

The molecular model is specified in terms of the type and position of each atom and the potential interaction between each pair of atom types. Point charges may also be included for each atom type. When using one of the standard potentials, it is possible to specify only like interactions (between atoms of the same type) and then interactions between unlike atoms determined from standard mixing rules.

PotentialMixing Rule
Hard Sphere σij = (σi + σj)/2
Soft Sphere σij = (σi + σj)/2 εij = (εi εj)1/2 n_ij = ni = nj
WCA σij = (σi + σj)/2 εij = (εi εj)1/2
Lennard-Jones σij = (σi + σj)/2 εij = (εi εj)1/2
Square Well σij = (σi + σj)/2 εij = (εi εj)1/2 λij = (σiλi + σjλj)/(σi + σj)

Alternatively, in the absence of standard mixing rules, the parameters for each pair of atom types must be specified. For even more flexibility, the "mixed" potential may be selected, which allows the potential (HS, LJ, etc) to be specified for each pair of atom types.
