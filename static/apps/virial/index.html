Virial Coefficients via Emscripten

emscripten compiles native code (C and C++) targetting Javascript (WebAssembly). The pages below use this to compute virial coefficients for simple spherical models.

B2 compute B2 using quadrature or a series expansion.

B3 compute B3 using fast Fourier transforms (FFT).

MSMC compute any order coefficient using Mayer-Sampling Monte Carlo.
