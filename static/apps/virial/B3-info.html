B3 can be computed for any spherical potential via fast-Fourier transforms (FFT). The computational cost of FFT calculations scale like N ln(N), where N is the number of distances where the interaction (and Mayer function) is evaluated. This is dramatically faster than using direct quadrature, which would scale like N3. The results, including the first two derivatives of the coefficient with invrese temperature (β=1/kBT) are reported, using N ranging from 25 to 216 to provide a measure of convergence.
