How this site works

The equations of state and analysis scripts on this site are written in Python. The site uses Brython, which allows Python 3 code to run in the browser and also provides the standard Python libraries. Computing the third virial coefficient (B3) on the fly is particularly intensive and too slow via Brython. Accordingly, the site uses a WebAssembly version of B3 written in C and built with emscripten.
